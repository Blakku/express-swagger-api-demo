# 📘 Express Swagger API Documentation Demo

This is a **sample REST API project built with Node.js, Express, TypeScript, and Swagger (OpenAPI 3.0)**.  
It demonstrates how to document backend APIs in a clean, interactive, and developer-friendly way using Swagger UI.

---

## 🚀 Features

- REST API built with Express.js
- TypeScript support for type safety
- Swagger/OpenAPI 3.0 integration
- Interactive API documentation (Swagger UI)
- Authentication endpoints (Register/Login)
- User management CRUD examples
- Request/Response examples for all endpoints
- Organized route structure

---

## 🧱 Tech Stack

- Node.js
- Express.js
- TypeScript
- Swagger UI Express
- Swagger JSDoc

---

## 📂 Project Structure

src/
 ├── app.ts              # Main server entry
 ├── swagger.ts          # Swagger configuration
 ├── data/
 │    └── db.ts         # Mock in-memory database
 └── routes/

      └── user.routes.ts
